🚖 DJ-Cabs – Travel & Cab Booking Website

DJ-Cabs is a simple and user-friendly travel and cab booking website that allows users to explore cab services, tour packages, and connect instantly for bookings.

⸻

🌟 Project Overview

The goal of this project is to provide an easy platform for users to book cabs and plan trips. It features a clean design and integrates WhatsApp for quick communication.

⸻

✨ Features
	•	🚕 Easy cab booking interface
	•	📦 Tour packages section
	•	🧭 Custom travel planning
	•	📱 WhatsApp direct contact button
	•	🌄 Attractive landing page
	•	📄 Simple and responsive design

⸻

🛠️ Tech Stack
	•	HTML
	•	CSS
	•	JavaScript
